Cards

A card, similar to a playing card, serves as an entry point to more detailed information. A card could contain a photo, text, and a link about a single subject.

Overview

Cards may contain a photo, text, and a link about a single subject. They may display content containing elements of varying size, such as photos with captions of variable length.

A card set is a layout of similar cards.

Width

            
By default cards are fluid and will expand to the width of thier container. Individual card width may be controlled responsivly by grid sizing or manually using a helper class. If the card is part of a group, column or deck widths are determined automatically. Card become 100% when the viewport size is XS

            


            
Card Sets

            


            
Card Groups

            
Use card groups to render cards as a single, attached element with equal width and height columns. Card groups use display: flex; to achieve their uniform sizing.

            


            
Card Decks

            
Need a set of equal width and height cards that arenâ€™t attached to one another? Use card decks.

            


            
Card Grids

            
A card grid, much like a card deck, will provide a set separated equal height cards. However, unlike a card deck, card grids can have multiple rows and may responsive grids for cards of varying sizes.

Classes

            

                
                    

                        Class
                        Description
                    

                
                
                    

                        .card
                        Defines a basic card
                    

                    

                        .card-header
                        Initializes card header
                    

                    

                        .card-block
                        Text
                    

                    

                        .card-footer
                        Initializes card footer
                    

                    

                        .card-title
                        Initializes card title
                    

                    

                        .card-subtitle
                        Initializes card subtitle
                    

                    

                        .card-text
                        Initializes card text
                    

                    

                        .card-link
                        Applies margin-left settings
                    

                    

                        .card-blockquote
                        Remove default margins from blockquotes
                    

                    

                        .card-img
                        Initializes card image
                    

                    

                        .card-img-top
                        Applies rounded corners for top positioning
                    

                    

                        .card-img-bottom
                        Applies rounded corners for bottom positioning
                    

                    

                        .card-img-overlay
                        Position content absolutely over top of an image
                    

                    

                        .card-*
                        Use modifier classes to change the color of each card by adding .card-* to the .cards class. You may also apply variations by adding .dk, .dker, .lt, .lter to darken or lighten the base color.
                    

                    

                        .card-stacked
                        Applies styling to create the appearance of being stacked
                    

                    

                        .card-group
                        Renders a single row of cards as an attached group with equal heights and widths.
                    

                    

                        .card-deck
                        Renders a single row of cards as an detached group with equal heights and widths.
                    

                    

                        .card-deck-wrapper
                        Removes default margins around the card deck.
                    

                    

                        .card-columns
                        Creates a staggered card group
                    

                    

                        .card-grid
                        Creates a responsive card grid. Use with col-* grids system to create a multi-row grid of equal heights and widths. Uses css3 flexbox model. Each card displays 100% for XS viewports.
