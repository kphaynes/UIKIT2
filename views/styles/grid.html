Bootstrap Grid System

We incorporate the Bootstrap grid system, so you can get the layouts as you want. Based on a 12 column system, each column size (col-sm-*) applies to any one of 5 viewports. See the bootstrap documenation for more information.

Custom Grid System

As an extention to the bootstrap grid system you can use a sub-grid divided by 5ths (12/5) in any of the available viewport sizes (.col-*-5, .col-sm-*-5, .col-md-*-5, .col-lg-*-5, .col-xl-*-5).

Gutter Size

By default the grid system uses a 1.875rem (30px) gutter. You can decrease the gutters by appending .row-sm to the class.row

No Gutter

The gutters between columns in our predefined grid classes can be removed with .no-gutters. This removes the negative margins from .row and the horizontal padding from all immediate children columns.
