Progress

Use progress component for displaying simple or complex progress bars.

Basic

Progress components are built with two classes .progress and .progress-bar to provide baseline styles for the progress bar.

Classes

            

                
                    

                        Class
                        Description
                    

                
                
                    

                        .progress
                        Defines the progress
                    

                    

                        .progress-bar
                        Indicate the progress so far. It requires an inline style, utility class, or custom CSS to set their width. Use role and aria attributes to make it accessible.
                    

                     

                        .bg-*
                        Use background utility classes bg-* to change the appearance of individual progress bars. 
                    

                    

                        .progress-bar-striped
                        Apply a stripe via CSS gradient over the progress bar’s background color
                    

                    

                        .progress-bar-animated
                        Add to .progress-bar to animate the stripes right to left via CSS3 animations.
