Buttons

Create a button with a <button> or <a> element to retain the native click function. Use a .disabled class when a button can’t be clicked.

Button Colors

The base .btn class defines the button element and looks like a plain text link. Adding a .btn-* class will define the button styling. In addition to the standard bootstrap symantic colors you may add a .btn-dark, .btn-black, and .btn-white 



            
Button Outlines

            
Use the .btn-outline-* class instead of .btn-* for a outline button. You can increse the border size using .border-width-2, .border-width-2, .border-width-4 or .border-width-5. In addition to the standard bootstrap symantic colors you may add a .btn-secondary, .btn-white, .btn-dark, and .btn-black.

            

            


            
Button Sizes

            
This example shows the various sizes a button can be acheived. Beyond bootstrap sizes you can also apply .btn-xl and .btn-xs 

            

            


            
Button Widths

            
Buttons are as wide as their text. For specific sizing use .btn-w-xl, .btn-w-lg, .btn-w-md, .btn-w-sm or .btn-w-xs 

            

            


            
Responsive Widths

            
Add a .btn-block to acheive a reponsive button. You may also leverage various sizes.

            

            


            
Rounded Buttons

            
Add a .btn-rounded class to acheive a rounded button

            

            


            
Icon Buttons

            
Add a .btn-icon class to acheive a icon button

            


            
Button Toggle

            
Add data-toggle="button" to toggle a button’s active state.
