Spark UI

A design system and component assetsrary, built using kendo and bootstrap for building custom progress applications.

Design Beautiful Applications Quicker

The Spark Design System is a collection of modern design patterns for building custom progress applications with a unified look and feel. It combines and extends the power of Kendo UI and Twitter Bootstrap and includes CSS, HTML, icons, color palettes, and fonts. Our vision is to provide a clear way to communicate simple, personalized, multi-device user experiences to engineering teams.

Build Consistently

Leverage established visual and interaction design patterns for a consistent experience. Consistency in UI and UX makes users jobs easier, reducing complexity and technical debt.

Improve Production Speed

With preset guidelines and a set of resources for all content, you can focus your energy on creating great workflows and logic, no UI.

Flexibility

In additiona to browser and mobile support the design system can be used with any technology stack you want. Additionally, we provide guidance around how to use it in conjunction Kendo UI.
